Visible Armor Slots Mod
=======================

Minecraft mod that makes the armor and off-hand slots visible on many GUIs.


Planned features 
--------------

Armor slots and off-hand slot visible on:

Blocks:

- [x] Enchantment Table;
- [x] Anvil;
- [x] Chest;
- [x] Trapped Chest;
- [x] Double Chest;
- [x] Trapped Double Chest;
- [x] Ender Chest;
- [x] Furnace;
- [x] Crafting Table;
- [x] Dropper;
- [x] Dispenser;
- [x] Hopper;
- [x] Brewing Stand;
- [ ] Beacon;

Entities:

- [ ] Minecart Chest;
- [ ] Minecart Hopper;
- [ ] Donkey;
- [ ] Mule;
- [ ] Horse;
- [ ] Villager;
