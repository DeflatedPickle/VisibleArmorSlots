Visible Armor Slots Mod
=======================

Minecraft mod that makes the armor and off-hand slots visible on many GUIs.


Planned features 
--------------

Armor slots and off-hand slot visible on:

- [x] Enchantment Table;
- [x] Anvil;
- [ ] Chest;
- [ ] Trapped Chest;
- [ ] Double Chest;
- [ ] Trapped Double Chest;
- [ ] Ender Chest;
- [ ] Dropper;
- [ ] Dispenser;
- [ ] Hopper;
- [ ] Furnace;
- [ ] Beacon;
- [ ] Brewing Stand;
- [ ] Minecart Chest;
- [ ] Minecart Hopper;
- [ ] Donkey;
- [ ] Mule;
- [ ] Horse;
- [ ] Villager;
