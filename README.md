Visible Armor Slots Mod
=======================

Minecraft mod that makes the armor and off-hand slots visible on many GUIs.


Planned features 
--------------

Armor slots and off-hand slot visible on:

- [x] Enchantment Table;
- [x] Anvil;
- [x] Chest;
- [x] Trapped Chest;
- [x] Double Chest;
- [x] Trapped Double Chest;
- [x] Ender Chest;
- [ ] Furnace;
- [ ] Crafting Table;
- [ ] Dropper;
- [ ] Dispenser;
- [ ] Hopper;
- [ ] Brewing Stand;

- [ ] Beacon;

- [ ] Minecart Chest;

- [ ] Donkey;
